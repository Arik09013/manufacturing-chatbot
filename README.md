# LLM Multimodal Manufacturing Chatbot — MVP

An operator-facing chatbot that answers natural-language questions about production anomalies by running a deterministic ML pipeline and using an LLM only to synthesize the result into plain English.

## Architecture

```
Operator question
       │
       ▼
  NL parsing (LLM intent)
       │
       ▼
  Pipeline: load → preprocess → fuse → detect → explain (SHAP) → reason → confidence
       │
       ▼
  LLM synthesis layer (Claude)
       │
       ▼
  Structured answer: anomaly + root cause + recommendation + SHAP explanation + confidence
```

## Modalities

| Modality | Description |
|---|---|
| Sensor time-series | Temperature, vibration, pressure, RPM, power per machine |
| Production logs | Event codes, maintenance events, warnings |
| Operator notes | Free-text shift notes |

## Quick Start

```bash
# 1. Create and activate virtual environment
python -m venv .venv
.venv\Scripts\activate  # Windows
# source .venv/bin/activate  # Linux/macOS

# 2. Install dependencies
pip install -r requirements.txt

# 3. Copy and fill environment variables
copy .env.example .env
# Edit .env and set ANTHROPIC_API_KEY=your_key

# 4. Generate synthetic dataset
python src/data/generate_synthetic.py

# 5. Run the Streamlit app
streamlit run app/streamlit_app.py

# 6. (Optional) Run the FastAPI backend separately
uvicorn src.api.main:app --reload
```

## Project Structure

```
manufacturing-chatbot/
├── data/
│   ├── raw/           # Synthetic CSV inputs (generated by T1)
│   └── processed/     # Fused parquet (generated by T4)
├── models/            # Trained anomaly detector (T5)
├── outputs/           # SHAP plots, eval reports
├── config/            # Rule maps, settings
├── src/
│   ├── data/          # Loaders, generators, schemas
│   ├── preprocess/    # Per-modality preprocessing
│   ├── fusion/        # Temporal alignment + feature fusion
│   ├── model/         # Anomaly detection
│   ├── explain/       # SHAP explainability
│   ├── reasoning/     # Root-cause + recommendation logic
│   ├── chat/          # LLM synthesis layer
│   └── api/           # FastAPI backend
├── app/               # Streamlit frontend
├── demo/              # Demo scripts and walkthrough
└── tests/             # Evaluation and unit tests
```
